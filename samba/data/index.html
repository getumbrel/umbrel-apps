How to Connect to the Umbrel SMB Server

Follow these steps to connect to the Umbrel SMB Server:

For Windows Users

  1. Open File Explorer.
  2. In the address bar, type \\umbrel.local and press Enter.
  3. When prompted, enter the username: umbrel and the password: changeme.
  4. You should now see your /storage directory.

For macOS Users

  1. Open Finder.
  2. In the menu bar, click on Go and then select Connect to Server....
  3. In the Server Address field, type smb://umbrel.local and click Connect.
  4. When prompted, enter the username: umbrel and the password: changeme.
  5. You should now see your /storage directory.

For Linux Users

  1. Open a file manager (such as Nautilus or Dolphin).
  2. In the address bar, type smb://umbrel.local and press Enter.
  3. When prompted, enter the username: umbrel and the password: changeme.
  4. You should now see your /storage directory.

If you encounter any issues, make sure that the Umbrel SMB Server is running and that you have the correct permissions to access it. Additionally, ensure that your firewall or antivirus software is not blocking the connection.
