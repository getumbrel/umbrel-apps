Your local Hermes Agent is up.

Hermes Agent is running on your Umbrel as a private local service. Drop into the app terminal for the native CLI, or connect Hermes Workspace to this same backend for a full UI.

Option 1 · Use the Hermes CLI

Open the app shell from umbrelOS, connect your model, then continue the whole session inside Hermes.

  1. Go to Settings -> Advanced Settings -> Terminal -> App.
  2. Select Hermes Agent.
  3. Set your AI model by running:
hermes model
  1. Once your model is connected, start chatting with your agent by running:
hermes
  1. From here, you can ask your agent for eg. how to talk to you via telegram? and continue.

Option 2 · Use Hermes Workspace

Install Hermes Workspace from the Umbrel App Store to work with this running Hermes backend through a web UI.

Same backend
Workspace connects to the Hermes instance already running here.
Private by default
Everything starts local, with no default public API exposure.
Configurable API access
If you need other API clients, enable and secure that access inside Hermes for your setup.
